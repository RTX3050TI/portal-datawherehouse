Acerca del Sistema SEGEP

Arquitectura estructural y créditos del equipo de desarrollo.

Arquitectura y Tecnologías

El Sistema de Evaluación de Gasto (SEGEP) es una plataforma integral de Inteligencia de Negocios diseñada para analizar la ejecución presupuestal de la emergencia COVID-19. La arquitectura se divide en tres capas fundamentales:

  • Data Warehouse (Azure SQL Server): Repositorio centralizado modelado en esquema estrella con 5 vistas maestras materializadas para consultas multidimensionales rápidas.
  • Capa de Visualización (Power BI): Dashboard corporativo para el análisis descriptivo, territorial y detección de ineficiencias mediante KPIs.
  • Motor de Inteligencia Artificial (AURA): API desarrollada en Java y alojada en Azure App Services, capaz de procesar lenguaje natural y aplicar modelos descriptivos, predictivos y prescriptivos.
